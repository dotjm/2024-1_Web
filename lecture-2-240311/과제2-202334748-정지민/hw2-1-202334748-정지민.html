   예 :


2. <meta> : 메타 정보를 정의
   예 :

<meta charset="UTF-8"> =>

3. <del> : 단어 중앙을 통과하는 줄을 생성
   예 :

<del>사과</del> => 사과

4. <h1> : 제목 글자 생성 (h는 heading을 의미, 1부터 6까지 숫자로 크기 조절 가능)
   예 :

<h1>사과</h1> =>

사과


5. <pre> : 개발자가 조절한 그대로 입력
   예 :

<pre>사과</pre> => 
사과

6. <b> : 텍스트를 진하게 표시
   예 :

<b>진하게</b> => 진하게

7. <strong> : 텍스트를 중요하게 강조, 보통 진하게 표시됨
   예 :

<strong>중요한</strong> => 중요한

8. <em> : 텍스트를 강조, 일반적으로 이탤릭체로 표시됨
   예 :

<em>강조</em> => 강조

9. <i> : 텍스트를 이탤릭체로 표시하여 강조
   예 :

<i>이탤릭으로 강조</i> => 이탤릭으로 강조

10. <dl> : 정의 리스트(definition list) 생성
   예 :

<dl><li>Apple</li><li>Banana</li></dl> =>
  • Apple
  • Banana

    • 11. <small> : 텍스트를 한 단계 작게 표시
       예 :

    보통 문자 <small>한 단계 작은 문자</small> => 보통 문자 한 단계 작은 문자

    12. <del> : 텍스트 중간에 줄을 그어 삭제 표시
       예 :

    <del>삭제</del> => 삭제

    13. <ins> : 텍스트에 밑줄을 그어 추가나 변경을 표시
       예 :

    <ins>추가</ins> => 추가

    14. <sup> : 텍스트를 윗첨자로 표시
       예 :

    보통문자의 <sup>윗첨자</sup> => 보통문자의 윗첨자

    15. <sub> : 텍스트를 아래첨자로 표시
       예 :

    보통문자의 <sub>아래첨자</sub> => 보통문자의 아래첨자

    16. <mark> : 텍스트 배경을 하이라이팅
       예 :

    <mark>하이라이팅</mark> => 하이라이팅

    17. <a> : 다른 페이지나 사이트, 문서 내 특정 부분으로 링크
       예 :

    <a href="https://google.com">Google 방문하기!</a> => Google 방문하기!

    18. <img> : 이미지를 삽입
       예 :

    <img src="media/apple.jpg" alt="Apple"> =>

    19. <ol> : 순서가 있는 목록 생성
       예 :

    <ol><li>First item</li><li>Second item</li></ol> =>
    1. First item
    2. Second item

    20. <ul> : 순서가 없는 목록 생성
       예 :

    <ul><li>Apple</li><li>Banana</li></ul> =>
    • Apple
    • Banana
